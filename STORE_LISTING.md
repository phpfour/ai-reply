# Chrome Web Store Listing

## Extension Name
**Smart Reply - AI-Powered Social Media Replies**

## Short Description (132 characters max)
Generate thoughtful, context-aware replies on LinkedIn, X (Twitter), YouTube & Instagram with AI. Uses your own OpenAI API key.

## Detailed Description

**Smart Reply** helps you craft engaging, authentic replies on social media in seconds. Powered by AI, it analyzes the post you're replying to and generates multiple context-aware suggestions that sound natural and human.

### Supported Platforms
- LinkedIn
- X (Twitter)
- YouTube Comments
- Instagram

### Key Features

**Context-Aware Replies**
Smart Reply reads the post content, author, and context to generate replies that actually reference what was said - no more generic "Great post!" comments.

**Multiple Suggestions**
Get 3 different reply options, each taking a unique angle - build on their point, ask a follow-up question, or share a related insight.

**Tone Control**
Choose from Auto, Friendly, Humorous, or Engaging tones to match your style and the conversation.

**Personalization**
Add your name, occupation, and background so replies reflect your expertise and personality.

**Custom Instructions**
Set specific guidelines like "keep replies under 280 characters" or "always mention my expertise when relevant."

**Natural Typing Simulation**
Replies are typed character-by-character to work seamlessly with React-based platforms.

**Privacy First**
- Uses YOUR own OpenAI API key - we never see it
- No data sent to our servers
- All settings stored locally on your device

### How It Works
1. Navigate to a post on any supported platform
2. Click on the reply/comment box
3. Click the âœ¨ Smart Reply button
4. Choose your preferred tone
5. Select from 3 AI-generated suggestions
6. Edit if needed and post!

### Requirements
- OpenAI API key (get one at platform.openai.com)
- Chrome browser

### Model Support
Works with GPT-5, GPT-4o, GPT-4, o1, o3, and more. Select your preferred model in settings.
